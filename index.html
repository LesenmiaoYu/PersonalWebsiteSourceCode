Personal Website

My personal website is more than a portfolio—it’s a reflection of my creativity, technical skills, and passion for exploring new tools. What started as a fun side project became a meaningful way to showcase my work and journey.

I designed, developed, and deployed every aspect myself, using tools like React, JavaScript, CSS, Blender, and 3.js. Blender introduced me to 3D modeling, while 3.js added interactive visuals, pushing me to learn and grow.

Challenges included my initial plans for complex animations and camera movements, which proved too ambitious. I adapted by creating an open, exploratory design where users can interact freely, resulting in a more intuitive experience.

The website was an exercise in persistence, from research and experimentation to hosting on GitHub Pages. The result is something I’m proud of—visually unique, functional, and truly mine.

Beyond the technical skills, this project taught me to think like a web developer, focusing on user experience and adaptability. It’s a living project I plan to expand, adding new features and reflecting my growth as a developer.

MBRL Dashboard

The MBRL Dashboard solves a critical problem: scattered data. Researchers and administrators at the USC Marshall Lab needed a centralized platform for visualizing and aggregating their data, and this dashboard became the solution, automating the process with API calls and other tools.

As the sole developer, I managed every aspect, from designing the architecture to deployment. This project marked my first experience writing backend endpoints and deploying a site using AWS, pushing me to learn new skills and adapt quickly.

The project relied on Flask, React, SQLite, and AWS EC2. Building the dashboard was straightforward, but deploying it on AWS was a steep learning curve. Documentation and persistence were key to overcoming these challenges.

The dashboard has made an immediate impact, saving researchers countless hours by streamlining data aggregation and visualization. Stakeholders were thrilled with the results, and its functionality has the potential to benefit the lab for years to come.

This project strengthened my technical skills and allowed me to grow as a project leader. While it’s still a work in progress, I’m proud of its success and plan to enhance it further with debugging and verification processes. It’s hosted on GitHub and stands as a testament to my growth as a developer.

ELC Roleplay Platform

The ELC Roleplay Platform was created to enhance interactive learning by offering a scalable, multi-user environment for roleplay exercises. Designed for the USC Experiential Learning Center, it facilitates seamless collaboration and scenario-based learning for students and educators.

As the frontend developer, I focused on building a dynamic, user-friendly interface using React to ensure smooth interactions and an intuitive design. This was a critical component for creating an engaging learning experience.

One of the key challenges was implementing responsive and interactive features to support real-time collaboration across multiple users. By leveraging React’s component-based architecture, I ensured the platform could handle complex interactions efficiently.

Hosted on DigitalOcean, the platform is a powerful tool that has transformed how roleplay exercises are conducted. It’s a testament to the potential of technology in education and my ability to design interfaces that enhance user experiences.

HOX Website

Eta Omega Chi (HOX) is a national Asian fraternity at USC, dedicated to fostering community through social bonding and professional development. As president, I led the creation of a static website to support our recruitment process, marking a pivotal project for our organization.

The website was built using HTML and served as a starting point for my journey into web development. It was also my introduction to Figma, where I explored the basics of UI/UX design to create a clean and functional layout that reflected HOX's values and mission.

This project taught me the fundamentals of web development and design while providing a platform to streamline recruitment and showcase our fraternity to potential members. It remains a significant milestone in my growth as a developer and leader.

Park Finder

Park Finder is an academic project that utilizes the National Park API to showcase information about national parks across the U.S. The platform allows users to explore parks with an intuitive and user-friendly interface.

As the frontend developer, I was responsible for designing and implementing the user interface, ensuring a seamless experience. This project also introduced me to web security concepts, where we focused on building secure and reliable applications.

We incorporated extensive testing using Gherkin and JUnit to validate functionality and ensure robustness. These practices strengthened the platform’s reliability and gave me valuable insights into professional software testing methodologies.

This project was a foundational experience that deepened my understanding of frontend development, testing strategies, and web security principles, preparing me for future real-world applications.

EAT! Web Application

EAT! is a web application designed to enhance the dining experience at USC by providing a platform where students can review and rate dining hall meals. With features like real-time reviews, personalized recommendations, and wait time updates, EAT! helps students make informed dining choices.

As a team member, I contributed to both frontend development and feature design, using HTML, CSS, and Figma to create a user-friendly interface. On the backend, the project utilized Java, Spring Framework, and MySQL to manage data securely and efficiently.

We incorporated advanced functionality such as multithreaded interactions for likes and dislikes, along with personalized user profiles. This project also introduced me to password encryption and database management, deepening my technical expertise.

EAT! was a rewarding experience that showcased the power of collaboration and innovation in solving real-world problems. It’s a practical tool that continues to serve the USC community by simplifying dining decisions.
